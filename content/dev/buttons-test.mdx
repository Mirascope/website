## Button Link Component

The ButtonLink component creates links styled as buttons.

```tsx
<ButtonLink href="#" variant="default">Internal Link</ButtonLink>
<ButtonLink href="https://example.com" variant="outline">External Link</ButtonLink>
```

<div className="flex flex-wrap gap-4 my-6">
  <ButtonLink href="#" variant="default">Internal Link</ButtonLink>
  <ButtonLink href="#" variant="outline">Outline Link</ButtonLink>
  <ButtonLink href="#" variant="ghost">Ghost Link</ButtonLink>
  <ButtonLink href="#" variant="link">Link Style</ButtonLink>
  <ButtonLink href="https://example.com" variant="default" external>External Link</ButtonLink>
</div>
