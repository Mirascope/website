# Code Block Length Test

This page tests code blocks with different lengths.

## 1 Line

```js
console.log("This is a one-line code block");
```

## 2 Lines

```python
def hello():
    print("This is a two-line code block")
```

## 3 Lines

```ts
function greeting(name: string): void {
  console.log(`Hello, ${name}!`);
}
```

## 4 Lines

```jsx
function Welcome(props) {
  return (
    <h1>Hello, {props.name}</h1>
  );
}
```

## 5 Lines

```css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
  padding: 1rem;
}
```

## 6 Lines

```bash
#!/bin/bash
# This is a 6-line bash script
echo "Starting script..."
ls -la
echo "Script complete!"
exit 0
```
