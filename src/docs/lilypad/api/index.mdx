# Lilypad API Reference

Welcome to the Lilypad API reference documentation. This section provides detailed information about the Lilypad SDK classes, methods, and interfaces.

## Core Modules

- **DataFlywheel** - Manage data collection and evaluation
- **Collector** - Collect data from various sources
- **Evaluator** - Evaluate data quality
- **Storage** - Store and retrieve data
- **Feedback** - Process user feedback

## Examples

### DataFlywheel

```typescript
import { DataFlywheel } from "@mirascope/lilypad";

const flywheel = new DataFlywheel({
  name: "product-qa",
  storage: {
    type: "local",
    path: "./data/product-qa",
  },
});

// Add data
await flywheel.collect({
  input: "What are the dimensions of the XYZ product?",
  output: "The XYZ product measures 10 x 5 x 2 inches.",
});

// Retrieve data
const data = await flywheel.getData();
```

### Evaluator

```typescript
import { Evaluator } from "@mirascope/lilypad";

const accuracyEvaluator = new Evaluator({
  name: "accuracy",
  prompt: "Rate the accuracy of this response from 1-10: {{response}}",
  responseModel: {
    score: "number",
    reasoning: "string",
  },
});

const result = await accuracyEvaluator.evaluate({
  response: "Paris is the capital of France.",
});

console.log(result.score); // e.g., 10
console.log(result.reasoning);
```
