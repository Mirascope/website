## Basic Provider Test

This is a simple test of provider-specific code snippets.

```python
from mirascope.core import openai

@openai.call("gpt-4o-mini")
def hello_world():
    return "Say hello!"

response = hello_world()
print(response.content)
```

Below we'll test various approaches for making this provider-aware.

## Approach 1: Use Backticks with Provider Variables

```python
from mirascope.core import $PROVIDER_PACKAGE

@$PROVIDER_PACKAGE.call("$PROVIDER_MODEL")
def hello_world():
    return "Say hello!"

response = hello_world()
print(response.content)
```
