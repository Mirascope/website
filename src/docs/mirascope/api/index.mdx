# Mirascope API Reference

Welcome to the Mirascope API reference documentation. This section provides detailed information about the Mirascope SDK classes, methods, and interfaces.

## Core Modules

- **Prompt** - Create and manage prompts for LLMs
- **Call** - Make calls to different LLM providers
- **Stream** - Handle streaming responses
- **Tool** - Define and use tools for LLM agents
- **ResponseModel** - Define structured outputs

## Examples

### Prompt

```typescript
import { Prompt } from "@mirascope/sdk";

const GenerateResponse = Prompt.create({
  systemPrompt: "You are a helpful assistant.",
  userPrompt: "Answer the following question: {{question}}",
});

const response = await GenerateResponse.call({
  question: "What is the capital of France?"
});
```
