# Getting Started with Mirascope

Mirascope is a powerful tool for building LLM applications. This guide will walk you through the basics of setting up Mirascope in your project.

## Installation

```bash
npm install @mirascope/sdk
```

## Basic Usage

```typescript
import { Mirascope } from "@mirascope/sdk";

const mirascope = new Mirascope({
  apiKey: process.env.MIRASCOPE_API_KEY,
});

// Create a basic prompt
const response = await mirascope.prompt("Tell me about frogs");
console.log(response.text);
```
