# Manual Highlight Test

This post tests manual code line highlighting with explicit comment markers.

## JavaScript Example

```javascript
function greet() { // [!code highlight]
  console.log("Hello, World!"); // [!code highlight]
} // [!code highlight]

greet();
```

## Python Example

```python
def greet(name):
    # This line is not highlighted
    print(f"Hello, {name}!") # [!code highlight]
    
    return "Done" # [!code highlight]
```

That's it for the manual highlight test!
